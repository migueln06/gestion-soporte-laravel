# 🛠️ Sistema de Gestión de Inventario para Soporte Técnico
Desarrollado con **Laravel 11**, **Bootstrap 5** y **DomPDF**.

### 🌟 Funcionalidades Clave:
* **Dashboard de Métricas:** Visualización en tiempo real de equipos pendientes y entregados.
* **Filtros Dinámicos:** Búsqueda optimizada por cliente o marca de equipo.
* **Reportes Profesionales:** Generación automática de comprobantes de ingreso en PDF.
* **Arquitectura Limpia:** Uso de Eloquent ORM y controladores estructurados.

### 🚀 Instalación:
1. Clonar repositorio.
2. `composer install`.
3. `php artisan migrate`.
